# Community Compass - Reference Repositories

This repo is being built as a community resource dashboard inspired by KindConnect, Delaware.gov resource layouts, and an interactive service-first experience.

## Repositories to borrow code and ideas from

- https://github.com/niciahrymer-hillian/HomeMatch.git
  - Housing matching UI patterns
  - Listing detail cards and eligibility scoring
  - Map/listing interaction layout

- https://github.com/ShockaHolmes/Future-Path.git
  - Youth pathway and risk assessment flows
  - Progress step UI and caseworker connection interactions

- https://github.com/prachi0617/kindConnect
  - Dashboard patterns, hero category blocks, and resource navigation
  - Persistent assistant panel, notes, and saved item flows

- https://github.com/gitanitraj/FirstStep
  - Resource directory layout and state-style service listings
  - Accessibility-first design and card-based search experience

## AI provider plan

- Use GROQ as the AI response provider for contextual assistance and suggestions
- Use ElevenLabs for voice output and assistant narration once API keys are available
- Keep provider configuration in a secure environment file or vault

## Current scope

- Static prototype page with mobile-friendly dashboard, housing search, resource directory, news feed, youth pathway, reminders, and floating assistant panel
- Persistent notes saved locally in the browser
- Interactive mock behaviors for listing details, note saving, and AI suggestion panels
