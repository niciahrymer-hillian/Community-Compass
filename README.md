# Community Compass

Community Compass is a community resource dashboard prototype for housing, youth services, resource directories, and AI-assisted support.

## What’s included

- Dark navy + white + green interface inspired by KindConnect and state resource pages like Delaware.gov
- Sticky navigation with anchor links to each section
- Accessibility controls for font size and contrast
- Hero category blocks for Housing, Essentials, Youth, and Updates
- Dashboard stats with urgent deadlines, nearby resource counts, and AI suggestions
- Interactive housing map mockup, filter pills, and listing details with schedule, save, and email actions
- Resource directory search and category tabs for shelter, food, transit, and health services
- News feed styled as state briefings and updates
- Youth pathway risk questionnaire and caseworker connection flow
- Reminders and next-step action plan
- Persistent floating assistant panel with tabs for AI suggestions, notes, and saved items

## Files

- `index.html` — prototype page
- `styles.css` — layout and palette styling
- `script.js` — interactive UI behaviors
- `pull-repos.md` — reference roadmap for borrowing from existing repos and plans for AI providers
- `design/community_compass_architecture_uml.svg` — UML architecture asset
